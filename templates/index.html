Enhance Your Business Performance

Employee Emotion recognition System is a software that detects an emplyee's emotions through facial recognition then reports whether the employee is eligible to work or not for a particular task in a particular moment.

Manage Employees Leaves

This system is equiped with a Leave management sector, with both Admin and Employee login sector. An employee makes a request for a leave by describing the start date, end date and type of leave they wish to have. The admin (employer/manager/supervisor) receives the notification and either Approves or Declines the leave request.

Schedule Tasks

This system is also equiped with a task management sector, with both Admin and Employee login sector. The admin allocates tasks to each and every employee and views the task progress of each employee. Employee can only update their task progress.
